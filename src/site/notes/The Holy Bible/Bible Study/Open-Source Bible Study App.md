I've been using Logos as my primary [[The Holy Bible/Bible Study/Bible Study\|Bible Study]] app for a long time.  I still love it and will use it as long as it remains affordable for me.  This is one of the only subscriptions I pay for that I use multiple time each day, and will continue with the subscription as long as I am able to.  The only trouble that I have with it is that it does not run natively on Linux, and Windows is an operating system I've been trying to get away from.  [[I Hate Windows\|I Hate Windows]] for lots of reasons. I can still use Logos functionally online using the web version.  But offline, I'm restricted to using the android version on my phone, or continuing to [[Tech Notes/Dual-Boot\|Dual-Boot]] my laptop into windows to use Logos.

So, seeing as I have time on my hands, I'm developing and [[Tech Notes/Open-Source\|Open-Source]] bible study app while I continue to use Logos as my everyday study tool.  

The first thing I had to do is decide what translation of the Bible to use for the core document of my app.
