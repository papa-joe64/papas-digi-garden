CS focuses on understanding algorithms and data structures, software engineering including designing, testing and developing software systems, and computer architecture - understanding how hardware functions and developing code to utilize those hardware functions. 

Folks on the CS track study mathematics, logic, engineering and sprinkle in more than a little bit of creativity. 

CS creates. IT uses what has already been created.
